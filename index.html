Product Introduction

Product Overview

zett.ai is built around a server-class RISC-V CPU + NPU and can be paired with CXL memory expansion and LPU acceleration to create a coordinated CPU, NPU, CXL, LPU, and high-speed networking platform. It is designed for data-center and cloud workloads that need large memory capacity, high-bandwidth I/O, low-latency acceleration, and local deployment.

Flagship

RISC-V CPU + NPU AI Server Platform

The server-class processor targets next-generation data-center and cloud environments. It uses a 32-core 64-bit RISC-V architecture, runs at up to 3.0GHz, and combines DDR5, PCIe Gen5, virtualization, RAS, a hardware security engine, and a programmable NPU acceleration cluster. The result is a platform that covers general compute, AI inference, vector workloads, cloud-native virtualization, and trusted boot.

At the platform level, a CXL/LPU accelerator card can be attached as a programmable acceleration and large-memory expansion node. It provides PCIe Gen5 x16, CXL, DDR4 RDIMM, M.2 SSD, and QSFP-DD networking interfaces for real-time analytics, in-memory databases, AI/ML, HPC, financial modeling, and networking or storage offload workloads.

32-core RISC-V DDR5 6400MT/s PCIe Gen5 x64 64TOPS INT8 CXL + LPU
  • 32-core 64-bit RISC-V CPU, up to 3.0GHz, with single-core SPEC2006 greater than 16/G.
  • 64KB L1 instruction cache, 64KB L1 data cache, 1MB L2 cache, and 32MB L3 cache.
  • Supports RVA23, RISC-V virtualization extensions, AIA-1.0, RVV 1.0, and virtualized DMA access with IOMMU.
  • Four DDR5 subsystems up to 6400MT/s, up to 2TB per subsystem, with ECC and RDIMM/UDIMM/SODIMM support.
  • Four PCIe Gen5 controllers, up to 64 PCIe Gen5 lanes, and up to 32GT/s per lane.

Built for AI and Data-Intensive Workloads

The platform supports GenAI/LLM, embedded AI, in-memory databases, real-time feature stores, big-data analytics, model training, and edge inference. When datasets need to stay resident in memory, be retrieved quickly, or be processed close to the accelerator, the RISC-V CPU DDR5 subsystem and the CXL/LPU card's onboard memory and storage provide a more direct data path.

Integration and Manageability

The RISC-V CPU platform provides REPI-1.0 RAS, trace, debug, secure boot, hardware cryptography, WorldGuard security-domain isolation, and secure firmware OTA. The CXL/LPU card adds BMC management for temperature, voltage, current, and power monitoring, making it practical for server, workstation, and reconfigurable AI platform integration.

Key Advantages

Server-Class RISC-V Performance

32-core 64-bit CPU, up to 3.0GHz, a full cache hierarchy, and PCIe Gen5 I/O for cloud compute and high-concurrency workloads.

Large Memory and Fast Interconnects

DDR5 up to 6400MT/s and up to 2TB per subsystem; the CXL/LPU card extends the data path through CXL/PCIe and onboard memory.

AI and Vector Acceleration

The NPU supports INT, FP, and BF16 formats for LLM inference, image generation, vector compute, and matrix operations.

Security, RAS, and Operations

CE/UC/UCE error handling, trusted boot, hardware cryptography, trace/debug, and BMC monitoring support production deployments.
